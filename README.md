# 1-D-radar-drivers-RA129C-HLK-LD2451H-HLK-LD2415H-
This repository contains the complete driver implementation for a 1-D radar sensor for vechicle  speed detection. It is written  for real-time vehicle speed measurement , radar configuration and  processing algorithm to track the vehicle with the greatest speed in a traffic flow.
